Setup:
Arduino Mega 2560/R3
2.42 inch OLED Display / Pins: CS=10, DC=9, RS=8, SDA=51, SCK=52
Joystick with button (when pressing the joystick down) / Pins: VRX=A0, VRY=A1, SW=2
2x Pull down buttons / Pins: 12 & 13
Active Buzzer / Pin: 40hb c xf
   
   

Project:
A gameboy type of device with a homescreen where you can choose between a variety of fun games.

The Project should be logically separeted into multiple files. (E.g. the homescreen menu or a specific game have their own files)

Firstly there should be a homescreen where you can choose between different games.

Please write the code in a way where theres the possibility of adding more games later on.

For now, please create a fairly simple space invaders type game. Create a short menu with the previous high score shown and the options to start a game and to return to the menu. For the game: Create enemies that fly into the frame from the top. Create a few different enemies (non shooting ones, single shot ones and multi shot ones, ...). If an enemies shot or ship hits your ship you lose one life. You have three lives which are shown in the top left corner. There is also a score shown on the top right. The score increases with time and enemies killed. The player has a ship that can shoot single shots. There is also a small chance for a new life that the player can collect to appear. Enemies get increasingly faster, more and difficult but slowly. Please use the joystick for movement and one of the pull down buttons for shooting (with the ability to hold down the button for continuous shots). The other pull down button pauses the game and gives a pause screen ui with the options resume, menu, restart.

Create one more game which is like the dinosaur game in the firefox browser when you have no internet connection. Simple jump and run with obstacles and a score shown. For this game also a homescreen with previous highscore, start, and home as options. here just use one pull down button for jumping. and the other one as a pause screen again. the character jumping should be in the design of a phallus.
